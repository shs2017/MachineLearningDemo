Welcome to Jackson's Machine Learning Demo

This is page is a demo of some of the applications of machine learning code I've written. Below each section is a demonstration of individual cases of each.

Logistic Regression combined with Image Processing for Near-Pixel Prediction

In this I used a Logistic Regression (rounded at the end of computation) along with pre-computational image procession to predict what parts of the images fit the training data. In this specific example I used this to train the algorithm to predict what parts of an image are grass and which are not. To do this I gave it training data (pictures of grass and pictures of things without grass and labeled them respectively) and the algorithm then came up with parameters which allowed me to determine what part of an image (any image -- training data or not) has grass. Below is an webapp I wrote that allows you to upload your own file and the algorithm will output what pixels it "thinks" is grass using the previous black and white to indicate it. Note: if you put an image that is mainly grass the image will be mostly white so it might be hard to see. Also only png and jpg files are accepted if the file is jpeg or anyother extension please change it to jpg or png.

Convolutional Neural Network for Image Classification

In this I used a artificial neural network algorithm (trained via back-propigation) to classify an image as being in a city or rural setting. To do this I also wrote a neural network configuration file reader which takes a json file with a specific format and data to allow even greater generality (i.e. it can be used for any other situation requiring an artifical neural network)

Like before I have written a webapp that allows you to upload a file and the algorithm will output whether it "thinks" the image is in a rural or city setting along with the probrability thereof.
